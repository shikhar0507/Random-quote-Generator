# Random Quote Generator

This application generates random quote on click of a button. I created this application as a web development project for Free Code Camp

Technologies Used

For the making of this application, I used HTML/CSS/BOOTSTRAP for styling and making the webpage. The whole functionality is build using javascript and jquery.

# How does it work

On click of a button, a random quote appears on the screen. The quote is selected from a pre-defined set of Quotes in the array. Jquery manages the triggers as well as animation that goes with it.
